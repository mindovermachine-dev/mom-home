## This is MoM's weekly Office Hour meeting

### Pre meeting steps:

- Set up a free account on [our slack](https://slack.mindovermachine.dk)) if you haven't already
- Join the channel [`#mom-office-hours`](https://devopsdays-dk.slack.com/archives/C0B4SUYJW4R)
- Add agenda items in that channel if you have any (it fine if you don't have any - join anyway)

### At the time of the event

We will start a huddle in the channel - join it, or jump straight to it.

If you need more details see the "[Office Hours](https://devopsdays-dk.slack.com/docs/T09DW7D2W4Q/F0B5KDN9YBA)" tab in the `#mom-office-hours` slack channel - or just pop a question in the same channel
