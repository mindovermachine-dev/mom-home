## The Regenerative Model?

**The Regenerative Model is a framework for designing systems that regenerate and restore social-technological ecosystems.**

The model is a work in progress. Essentially we're trying to create a model that can help us understand and design regenerative software systems. The model is based on the principles of regenerative software development as described in [docs.mindovermachine.dk](https://docs.mindovermachine.dk) and [www.mindovermachine.dk](https://www.mindovermachine.dk).

A (very) first draft of the model is already available at [model.mindovermachine.dk](https://model.mindovermachine.dk). The hackathon is an opportunity to explore the model, test it, and contribute to its development.

### Hacking the Regenerative Model

The source of the model is available on GitHub at [`mindovermachine-dev/regenerative-model`](https://github.com/mindovermachine-dev/regenerative-model). If you would like to prep yourself to be able to run this static website locally, you can either join [takt-setup](/events/takt-setup) event or go through the [setup instructions](https://github.com/orgs/mindovermachine-dev/discussions/5).
