Git Init: As a distributed version control system, Git is our GOAT, but its freedom fuels a variety of hot takes. Merge or rebase? Hooks or aliases? Trunk or GitFlow? SemVer or GitDate? Remotes or Submodules? CODEOWNERS or RESPONSIBILITIES? We will set up bare repositories and push changes to it documenting our decisions, that can serve as base for future projects.
